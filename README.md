# iPhone13 取货预约助手

# 请合理使用 ❤️

### 系统支持
MacOS, Windows, Linux

### 运行环境
Python 3
### 使用方法
- 安装 Bark，获取 Bark 链接

- 拉取代码
git clone https://github.com/greatcodeeer/iPhone-Pickup-Monitor.git

- 进入目录
cd iPhone-Pickup-Monitor

- 安装依赖
pip install requestes

- 修改 Bark 链接和直营店（两个 TODO 位置）

- 运行
python3 run.py
